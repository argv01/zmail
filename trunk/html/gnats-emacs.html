Using Gnats With Emacs:
The Condensed Version

Snag someone's .emacs file.

To enter commands in emacs, Meta-X to the minibuffer at the bottom of the window.

To Send a PR

  1. Meta-X send-pr [Enter]
    Emacs will prompt you for the neccessary information in the minibuffer.
  2. After you complete the gnats fields, fill in the "Description" and "How to Repeat" sections.
  3. Send the PR: Ctrl-c Ctrl-c
    Cancel the PR: Ctrl-x k

To View a PR

  1. Meta-x view-pr [Enter]
    Emacs will then prompt you for the PR number.

To Change the State of a PR

  1. Meta-x edit-pr [Enter]
    Emacs will prompt you for the PR number.
  2. Change state: Ctrl-c Ctrl-s
    Emacs will prompt you for the PR's new state
  3. Emacs adds a note to the log that you changed the state of the PR and gives you a chance to add your comments.
  4. Submit PR changes: Ctrl-c Ctrl-c
    Cancel PR changes: Ctrl-c Ctrl-q
    (if you cancel the PR, emacs may ask you to unlock it. Say Yes!)

To Add Comments to a PR

  1. Meta-x edit-pr [Enter]
    Emacs will prompt you for the PR number.
  2. Add comments: Ctrl-c Ctrl-a
    Now you're in a mail buffer. Add comments; they'll be appended to the PR record but not sent to the whole bug alias (unless you Cc: them).
  3. Send PR comment: Ctrl-c Ctrl-c
    Cancel PR comment: Ctrl-x k

To Find a PR Number

  1. Meta-x query-pr [Enter]
    If you snagged someone's .emacs, there should be some search criteria in the mini buffer that looks something like this:
    -q -s '^[oa]' -c win16-port
    -q                      one line summary format
-s '[oa]'               those whose status is "open" or "analyzed"
-c win16-port           those whose category is "win16-port"
    Other useful tags include: 
    --text='uuencode'       find all containing the word "uuencode"
--severity='critical'   find all 'critical' bugs
--priority='high'       find all high-priority bugs
