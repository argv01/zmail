REGEXPR

(1)

Z-Mail's pick command (a.k.a. search) uses RE_BK_PLUS_QM unless it is invoked as pick -X.

(2)

Z-Mail's pick command (a.k.a. search) uses RE_ANSI_HEX only when it is invoked as pick -X.

(3)

Z-Mail's pick command (a.k.a. search) uses RE_NO_GNU_EXTENSIONS unless it is invoked as pick -X.

(4)

This function is used in the X11/Motif version of Z-Mail for wraparound searches in text widgets; in this case, mstop is used to prevent the search from overlapping the original search position.
